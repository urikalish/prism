Items

The item component is how a row is divided, It can host both content & other grid components.

- An item's width is determined by default by its content.
- An item has default padding in both axis.
- Items are stacked by default to the left (to the right when RTL).
.item 
<div class="container">
  <div class="row">
    <div class="item">
      ...
    </div>
    <div class="item">

    </div>

</div>

Each grid component can align its content by rows or columns.

  </div>

  </div>
</div>

- Item Sizes

An item can have different sizes like:
- An item's size can be determine only by its content.
- An item can take from 1 to 8 units of each row.
- An item can take the remaining space in a row dynamically.
- By default, all items in the same row are stretched to the same height.
- When items fill a row with more than 8 units, the remaining items are stacked to the bottom.
.size-{{i}}
the size of this item is determined by its content.
