Checkboxes

The checkbox component is used for boolean inputs

- Checkboxes can gain focus when tab is pressed.
- Checkboxes can respond to both spacebar & enter keys.
- Checkboxes can have 1 of 3 unique styles.
- 2-Way binding can be done with [(toggleVariable)]="someVar"
- Checkboxes can emit when their state was changed with (change)="someMethod($event)" where $event holds the current state
Default flavor: Checkbox
