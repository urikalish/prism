Filterable list

- An option to search across all list items
Searchable list settings:
  • placeholder option to set a static placeholder
  • hideList option to hide list if no search string is typed yet, default is no hidden
  • items list of items to display in the list
  • queryKey option to define the searchable key: item[queryKey], by default, item itself will be the searchable key
  • focus whether to focus on searchable input field, default is false
  • onItemSelected an option to implement onItemSelected event using (onItemSelected)="onItemSelectedImpl($event)"
Filterable list contains item.value as a searchable key: 
      <prism-filterable-list
          [items]="listOfItems"
          [queryKey]="'value'"
          [focus]="true">
      </prism-filterable-list>
