Rows

The row component defines a full-width row in the container, A row's width is composed of 8 equal units.

- A row has default padding on the left & right.
- A row has a bottom-margin unless it's the last of it kind.
- A row is divided to 8 equal size units.
- A row can align its item children in any direction.
