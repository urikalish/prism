Inputs

Input text field with 2 way binding model

- Input field has an option to receive a placeholder
- Input field can be filled with any content
- 2 way binding model can be done via [(ngModel)]="dataModel"
Input text field settings:
  • placeholder option to set a static placeholder
Input with placeholder: 
    <prism-input [(ngModel)]="dataModel">
    </prism-input>
Input with placeholder and two buttons on each side: 
{{ dataModel.text1 }}
<prism-input placeholder="Please enter text" [(ngModel)]="dataModel">
  <prism-button
      [border]="false"
      size="small"
      text=""
      image="someImage"
      (click)="onClickedImpl()">
  </prism-button>
  <prism-button
      [border]="false"
      size="small"
      text=""
      image="someImage"
      (click)="onClickedImpl()">
  </prism-button input-end>
</prism-input>
