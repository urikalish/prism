Dropdowns

Dropdown menu is a toggleable menu that allows the user to choose one value from a predefined list

- Dropdown has a toggle arrow button which indicates menu is opened\closed
- The list can be filled with any content
- The list can have either a displayed label or a selected value
- The user can implement item selected behaviour by emitting the click event (click)="onItemSelectedImpl(item.value)"
Dropdown settings:
  • displayedLabel dropdown menu label
  • selectedIndex option to set the index of selected item, default index is 0
  • itemSelector option to set HTML tag which can be selected, default selector is "<li>"
List item settings:
  • label list item label
Dropdown with displayed label: 
  <prism-dropdown [displayedLabel]="'label'" prismListNavigation>
    <prism-list>
      <prism-list-item
        *ngFor="let item of items"
        [label]="item.label"
        (click)="onItemSelectedImpl(item.value)">
      </prism-list-item>
    </prism-list>
  </prism-dropdown>
Dropdown with selected item index: 
  <prism-dropdown [selectedIndex]="selectedIndex" prismListNavigation>
    <prism-list>
      <prism-list-item
        *ngFor="let item of items"
        (click)="onItemSelectedImpl(item.value)">
      </prism-list-item>
    </prism-list>
  </prism-dropdown>

Color Picker

- Act as a dropdown menu
- Receive a list of colors
Color Picker settings:
  • colors list of colors
  • selectedIndex option to set the index of selected item 
<prism-color-picker [colors]="colors"></prism-color-picker>
